# DevSecOps Pro - Microservice Component

This repository is a submodule of the main devops-pro platform. 

## Local Development
This service is fully profiled. To run it locally within the broader ecosystem, please navigate to the root devops-pro repository and run:

``bash
docker-compose -f docker-compose.yml -f docker-compose.dev.yml up --build -d
``

For detailed architectural information, setup instructions, and authentication details, please refer to the guides/ directory in the root repository.
